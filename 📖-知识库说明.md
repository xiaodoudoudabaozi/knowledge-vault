# 🧠 知识库说明

这是你的个人知识库，基于 Obsidian + OpenClaw 构建。

## 📁 文件夹结构

| 文件夹 | 用途 |
|--------|------|
| `00-日记` | 每日日记、想法、灵感 |
| `01-工作` | 工作相关笔记、会议记录、业务内容 |
| `02-AI资讯` | AI领域资讯、工具测评、行业动态 |
| `03-学习笔记` | 沉淀学到的知识、技能笔记 |
| `04-项目` | 具体项目记录和进展 |
| `05-资源库` | 收藏的好文章、素材、资料 |
| `06-归档` | 已完成或不再活跃的内容 |
| `99-草稿箱` | 临时文件，写完移到对应目录 |

## 🔗 与 OpenClaw 联动

OpenClaw 会自动整理 AI 会话记录到对应目录，保持知识库持续更新。
