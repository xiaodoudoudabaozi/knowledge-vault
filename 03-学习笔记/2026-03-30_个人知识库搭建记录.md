## 遇到的问题

### GitHub 直连超时
- **问题**：`git push` 连接 GitHub 超时（TCP 443）
- **解决**：使用 `ghfast.top` 镜像，同时配合 GitHub Token 完成认证

### Python Unicode 编码问题
- **问题**：Windows PowerShell 默认 GBK 编码，emoji 字符报错
- **解决**：设置 `$env:PYTHONIOENCODING="utf-8"`

## 关键要点

- Obsidian 是本地知识库前端，GitHub 是云端备份
- GitHub CLI (`gh`) 可以访问 GitHub，但 git CLI 不行（网络策略问题）
- 知识库的核心价值在于**持续沉淀**，而非一次性整理
- Skill 自动化是下一步重点：自动归档 AI 会话内容
